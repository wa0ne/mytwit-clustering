MyTwit merupakan aplikasi untuk melakukan clustering pengguna Twitter menggunakan algoritma k-means. Aplikasi ini dibuat untuk memenuhi tugas mata kuliah Data Mining pada Jurusan Sistem Informasi ITS Surabaya.

Petunjuk penggunaan:

- Download JPGraph dari jpgraph.net lalu ekstrak pada folder src-nya ke dalam folder MyTwit lalu rename menjad jpgrah.
- buka file config pada folder config lalu isikan data CONSUMER_KEY, CONSUMER_SECRET, ACCESS_TOKEN, dan ACCESS_TOKEN_SECRET yang Anda dapatkan dari developer twitter.
- buka file db pada folder config dan sesuaikan dengan koneksi database mysql Anda.
- import mytwit.sql ke dalam database mysql anda.
